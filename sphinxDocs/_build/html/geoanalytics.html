geoanalytics.analytics.cosineSimilarity(tableName1, tableName2)[source]
Description:

This function calculates the cosine similarity between two dataframes using the provided formula.

Cosine Similarity = Dot product of df1 and df2 / (Magnitude of df1 * Magnitude of df2)

Parameters:

  • tableName1 – The first data frame containing ‘x’ and ‘y’ columns (point co-ordinates)

  • tableName2 – The second data frame containing ‘x’ and ‘y’ columns (point co-ordinates)

Returns:

A new DataFrame with ‘x’ , ‘y’ and ‘cosine similarity’ columns

Return type:

DataFrame
